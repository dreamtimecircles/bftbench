pub fn load_from_str(source: &str) -> Result<Vec<Yaml>, ScanError>

Load the given string as a set of YAML documents.

The source is interpreted as YAML documents and is parsed. Parsing succeeds if and only if all documents are parsed successfully. An error in a latter document prevents the former from being returned.

§Errors

Returns ScanError when loading fails.
